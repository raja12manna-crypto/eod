# EOD Solitaire — Full React Implementation Plan ## Context Recreate the EOD Solitaire HTML/JS card game as a React app, then add a floating hamburger menu that opens a slide-in drawer with two pages: **Settings** (per-suit hue sliders + live preview cards) and **Design System** (all UI components catalogued). Fully responsive: mobile, tablet, desktop. --- ## Aesthetic Stance - **Font:** Montserrat (Google Fonts) — 400, 600, 700, 800, 900 - **Ground:** Dark felt table — `radial-gradient` from `#003554` to `#002841` + dashed grid overlay - **Palette:** Green `#49B653` · Blue `#00456E` · Red `#C00C0F` · Yellow `#F4BA0E` · Ink `#0d1b26` · Paper `#F5F3EE` - **Borders:** 2px dashed `rgba(255,255,255,0.25)` — signature EOD guide-line style - **Radius:** pill buttons (999px), cards (10% of card width), modals (22px) --- ## Files to create / modify | File | Action | | ---------------------- | ----------------------------------------- | | `src/styles/fonts.css` | Add Montserrat Google Fonts import | | `src/styles/theme.css` | Update tokens to EOD dark palette | | `src/app/App.tsx` | Full implementation (see structure below) | --- ## App.tsx Structure ### State - `gameState` — full Klondike state (stock, waste, foundations[4], tableau[7]) - `undoStack` — array of JSON snapshots - `moves`, `seconds`, `timerRunning` - `learned` — Set of card IDs read (persisted in localStorage) - `selected` — `{pile, index} | null` - `winShown` - `drawerOpen` — boolean - `drawerPage` — `"settings" | "design-system"` - `suitHues` — `{ circle: number, square: number, triangle: number, hexagon: number }` (0–360, default 0 = original color) - `cardFontSize` — number (0.8–1.4, multiplier, default 1.0) - `learnModal` — card data | null - `helpOpen` — boolean - `winOpen` — boolean ### Components (all in App.tsx as local components) 1. **``** — root, holds all state, renders everything 2. **``** — EOD logo + stats (time/moves/terms) + Undo/?/New Game buttons 3. **``** — fixed top-right, floating, opens drawer 4. **``** — slide-in panel from right, contains nav tabs (Settings / Design System) + content 5. **``** — 4 individual hue sliders (one per suit), font size slider;
